# Build Instructions

To build, just run the file `build.bat`.

By default, the build will send data to my server. This can be changed either in `/Services/SettingsService.cs`, or - if you already have the `.exe` — by passing the following arguments:

- `-h` - custom hostname/IP
- `-p` - custom port
